# 蝰蛇音效驱动挂载器(ViPERFXDriverMounter)
    以不修改system分区的方式"安装"蝰蛇音效(ViPERFX)的驱动

# 用法
    1.root
    2.安装任意版本的ViPERFX[2.5.0.0+]
    3.选择"挂载并重启媒体服务"
    4.打开ViPERFX

# 注意
    1.每次重启后需要重新挂载
    2.Android O 仍然存在一些问题 可以尝试删除文件 /vendor/etc/audio_effects.xml 后再试
    3.Flyme因系统策略问题已确认无效
